# Student Management System

## Project Description
A web-based Student Management System built using Flask and MySQL.  
It helps to manage student details, marks, departments, and generate reports.

## Features
- Login system (Admin / Staff)
- Add, update, delete students
- Marks management system
- Department-wise student tracking
- Analytics dashboard
- PDF report generation

## Tech Stack
- Python (Flask)
- MySQL
- HTML, CSS
- ReportLab

## How to Run

1. Clone the repository
   git clone https://github.com/kavisreeR/student-management-system

2. Install dependencies
   pip install -r requirements.txt

3. Create database
   student_db

4. Run the application
   python app.py

5. Open browser
   http://127.0.0.1:5000
