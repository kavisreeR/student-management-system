# 🎓 College Management System

## 📌 Features
- Student Management (Add/Edit/Delete)
- Department Filtering
- Marks Entry System (CN, DAA, TOC, WT, DBMS)
- Auto Register Number Generation
- Report with Total & Average
- Analytics Dashboard (Charts, Toppers)

## 🛠 Tech Stack
- Python (Flask)
- MySQL
- HTML + Bootstrap
- Chart.js

## ⚙️ Setup Instructions

1. Install dependencies:
pip install -r requirements.txt

2. Import database:
Run database.sql in MySQL

3. Update DB config in app.py

4. Run:
python app.py
